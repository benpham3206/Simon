# Simon

A Simon memory game for the Adafruit Circuit Playground Express, written in
CircuitPython.

## Install

1. Install CircuitPython and the Adafruit Circuit Playground library bundle on
   the board.
2. Copy `code.py` to the root of the `CIRCUITPY` drive.
3. Reset the board.

## Play

1. Press button A to choose one of four sequence lengths.
2. Press button B to start.
3. Repeat the color and tone sequence using the capacitive touch pads.

The four difficulty levels use sequences of 5, 8, 10, and 15 steps.
